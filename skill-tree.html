Prototype of the RPG skill-tree mechanic for Guided Learning. Each dot is a real lesson; finish one to unlock the next (glowing = available, 🔵 = mastered). Tap an available node to master it and earn XP. Hover/tap a node to see which lesson it is. (Mechanic only — not yet wired to the real lessons or progress.)
